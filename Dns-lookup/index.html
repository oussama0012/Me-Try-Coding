What's DNS Again?

DNS (Domain Name System) is like the phone book of the internet. It translates human-readable domain names into IP addresses that computers can understand.

DNS records include:

  • A records - Point domains to IPv4 addresses
  • AAAA records - Point domains to IPv6 addresses
  • MX records - Define mail servers for the domain
  • CNAME records - Create domain aliases
  • TXT records - Store text information and verification data
  • NS records - Define authoritative name servers

Every domain has DNS records. Even your favorite meme site.
